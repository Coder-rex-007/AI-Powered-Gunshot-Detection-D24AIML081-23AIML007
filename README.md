# AI Gunshot Detection System

This project is an AI-based Gunshot Detection System using Machine Learning and Raspberry Pi.

## Features
- Gunshot sound detection
- Audio preprocessing using MFCC
- TensorFlow/TFLite model
- Raspberry Pi integration
- Real-time detection support

## Technologies Used
- Python
- TensorFlow
- Librosa
- NumPy
- OpenCV
- Raspberry Pi

## Files
- gunshot_detection_v2.ipynb
- gunshot_detection_v3_finetuned.ipynb
- training_plot.png

## Future Scope
- Live audio detection
- Cloud alert system
- Mobile notifications
